# CollegeCourse (React + Tailwind CSS)

This project is now fully implemented using **React** and **Tailwind CSS**.

## Features

- Home page with application details
- Navigation for all main sections/pages
- Footer across the application
- Login page and Register page
- Default user auto-created on first run
- Protected dashboard after login
- Developer course list shown after successful login

## Default Login

- Email: `developer@collegecourse.com`
- Password: `dev12345`

## Run the App

```bash
npm install
npm run dev
```
